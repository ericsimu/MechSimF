# 接入系统登录用户指南

## 当前认证机制

项目目前使用**极简认证**：前端从 `localStorage` 读取 `current_user`（默认 `user1`），通过 HTTP 头 `X-User` 传给后端。后端从请求头提取用户名做权限校验（仅创建者可编辑/删除用例）。

```
前端 localStorage  →  X-User 请求头  →  后端 get_current_user()
```

**涉及文件**：

| 端 | 文件 | 关键代码 |
|----|------|----------|
| 前端 | `src/api/index.ts` | `getCurrentUser()` 读 localStorage，`request()` 自动带 X-User |
| 前端 | `src/views/CaseList.vue` | `buildCaseBody()` 用 `getCurrentUser()` 填充 `create_by` |
| 后端 | `app/api/v1/endpoint/sim.py` | `get_current_user()` 从请求头提取，`Depends()` 注入 |

## 方案一：JWT Token 认证（推荐）

适用场景：已有一套账号系统，需要完整的登录/登出/会话管理。

#### 1.2 新增 `app/core/auth.py`

```python
from datetime import datetime, timedelta
from typing import Optional
from jose import JWTError, jwt
from passlib.context import CryptContext
from fastapi import Depends, HTTPException
from fastapi.security import OAuth2PasswordBearer
from app.config import settings

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")
oauth2_scheme = OAuth2PasswordBearer(tokenUrl=f"{settings.API_V1_PREFIX}/auth/login")

def create_access_token(username: str, expires_delta: Optional[timedelta] = None) -> str:
    expire = datetime.utcnow() + (expires_delta or timedelta(minutes=30))
    return jwt.encode({"sub": username, "exp": expire}, settings.SECRET_KEY, algorithm="HS256")

def verify_token(token: str) -> str:
    try:
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=["HS256"])
        return payload.get("sub", "")
    except JWTError:
        raise HTTPException(status_code=401, detail="无效的令牌")

def get_current_user(token: str = Depends(oauth2_scheme)) -> str:
    return verify_token(token)
```

#### 1.3 新增登录接口 `app/api/v1/endpoint/auth.py`

```python
from fastapi import APIRouter, Depends, HTTPException
from fastapi.security import OAuth2PasswordRequestForm
from pydantic import BaseModel
from app.core.auth import create_access_token, pwd_context

router = APIRouter()

# 模拟用户表（实际项目替换为数据库查询）
USERS = {"admin": pwd_context.hash("admin123"), "user1": pwd_context.hash("user1")}

class TokenResponse(BaseModel):
    access_token: str
    token_type: str = "bearer"
    username: str

@router.post("/auth/login", response_model=TokenResponse)
async def login(form_data: OAuth2PasswordRequestForm = Depends()):
    hashed = USERS.get(form_data.username)
    if not hashed or not pwd_context.verify(form_data.password, hashed):
        raise HTTPException(status_code=401, detail="用户名或密码错误")
    token = create_access_token(form_data.username)
    return TokenResponse(access_token=token, username=form_data.username)
```

### 2. 前端改造

#### 2.1 新增 `src/api/auth.ts`

```typescript
const AUTH_KEY = 'mechsim_token'
const USER_KEY = 'mechsim_user'

export function getToken(): string {
  return localStorage.getItem(AUTH_KEY) || ''
}

export function getUser(): string {
  return localStorage.getItem(USER_KEY) || ''
}

export function setAuth(token: string, username: string): void {
  localStorage.setItem(AUTH_KEY, token)
  localStorage.setItem(USER_KEY, username)
}

export function clearAuth(): void {
  localStorage.removeItem(AUTH_KEY)
  localStorage.removeItem(USER_KEY)
}

export function isLoggedIn(): boolean {
  return !!getToken()
}
```

#### 2.2 改造 `src/api/index.ts` — request() 函数

```typescript
import { getToken } from './auth'

async function request<T>(url: string, options: RequestInit = {}): Promise<ApiResponse<T>> {
  const token = getToken()
  const headers: Record<string, string> = {
    'Content-Type': 'application/json',
    ...options.headers as Record<string, string>,
  }
  // JWT 认证
  if (token) {
    headers['Authorization'] = `Bearer ${token}`
  }
  // 保留 X-User 用于权限控制
  const user = getUser()
  if (user) {
    headers['X-User'] = user
  }
  // ... 其余不变
}
```

#### 2.3 新增登录页面 `src/views/Login.vue`

```vue
<template>
  <div class="login-page">
    <form @submit.prevent="handleLogin" class="login-form">
      <h2>MechSim 登录</h2>
      <input v-model="username" placeholder="用户名" />
      <input v-model="password" type="password" placeholder="密码" />
      <button type="submit" :disabled="loading">登录</button>
      <p v-if="error" class="error">{{ error }}</p>
    </form>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { useRouter } from 'vue-router'
import { setAuth } from '../api/auth'

const router = useRouter()
const username = ref('')
const password = ref('')
const loading = ref(false)
const error = ref('')

async function handleLogin() {
  loading.value = true; error.value = ''
  try {
    const form = new FormData()
    form.append('username', username.value)
    form.append('password', password.value)
    const res = await fetch('/api/v1/sim/auth/login', { method: 'POST', body: form })
    const data = await res.json()
    if (res.ok) {
      setAuth(data.access_token, data.username)
      router.push('/cases')
    } else {
      error.value = data.detail || '登录失败'
    }
  } catch { error.value = '网络错误' }
  finally { loading.value = false }
}
</script>
```

## 方案二：SSO / OAuth2 对接（企业微信/飞书/统一认证平台）

适用场景：需要对接企业内部统一认证系统。

### OAuth2 授权码流程

```
用户点击登录 → 跳转SSO授权页 → 用户授权 → 回调带code
→ 后端用code换token → 后端返回JWT → 前端存储JWT
```

### 后端新增 `app/core/sso.py`

```python
import httpx
from fastapi import APIRouter
from app.core.auth import create_access_token

router = APIRouter()

SSO_CLIENT_ID = "your_client_id"
SSO_CLIENT_SECRET = "your_client_secret"
SSO_AUTH_URL = "https://sso.company.com/authorize"
SSO_TOKEN_URL = "https://sso.company.com/token"
SSO_USERINFO_URL = "https://sso.company.com/userinfo"
REDIRECT_URI = "http://localhost:5173/callback"

@router.get("/auth/sso/login")
async def sso_login():
    """重定向到 SSO 授权页"""
    from urllib.parse import urlencode
    params = {
        "client_id": SSO_CLIENT_ID,
        "redirect_uri": REDIRECT_URI,
        "response_type": "code",
    }
    from fastapi.responses import RedirectResponse
    return RedirectResponse(f"{SSO_AUTH_URL}?{urlencode(params)}")

@router.post("/auth/sso/callback")
async def sso_callback(code: str):
    """用授权码换取用户信息，返回 JWT"""
    async with httpx.AsyncClient() as client:
        # 1. code 换 access_token
        r = await client.post(SSO_TOKEN_URL, data={
            "client_id": SSO_CLIENT_ID,
            "client_secret": SSO_CLIENT_SECRET,
            "code": code,
            "grant_type": "authorization_code",
        })
        token_data = r.json()
        # 2. 用 access_token 获取用户信息
        r = await client.get(SSO_USERINFO_URL, headers={
            "Authorization": f"Bearer {token_data['access_token']}"
        })
        user = r.json()
    # 3. 签发自有 JWT
    jwt_token = create_access_token(user["username"])
    return {"access_token": jwt_token, "username": user["username"]}
```

### 前端 SSO 回调页面

```vue
<!-- src/views/Callback.vue -->
<script setup lang="ts">
import { onMounted } from 'vue'
import { useRoute, useRouter } from 'vue-router'
import { setAuth } from '../api/auth'

const route = useRoute()
const router = useRouter()

onMounted(async () => {
  const code = route.query.code as string
  if (!code) { router.push('/login'); return }
  const res = await fetch('/api/v1/sim/auth/sso/callback', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ code }),
  })
  const data = await res.json()
  setAuth(data.access_token, data.username)
  router.push('/cases')
})
</script>
```

## 方案三：反向代理注入（最简单）

适用场景：前端部署在 Nginx/Apache 后面，Web 服务器已做认证，只需透传用户名。

### Nginx 配置

```nginx
location /api/ {
    auth_request /auth;
    proxy_set_header X-User $remote_user;   # 或从 SSO header 获取
    proxy_pass http://localhost:8000;
}
```

后端无需改动，已有的 `get_current_user()` 直接从 `X-User` 头读取。前端也无需改动，直接移除 `localStorage` 逻辑即可。

## 各方案对比

| 方案 | 改造量 | 安全性 | 适用场景 |
|------|--------|--------|----------|
| JWT Token | 中 | 高 | 自建账号系统 |
| SSO/OAuth2 | 大 | 最高 | 企业统一认证 |
| 反向代理 | 小 | 依赖代理 | 已有网关认证 |

## 核心接入点总结

无论采用哪种方案，最终只需确保两件事：

1. **后端**能拿到可信的用户标识（`get_current_user()` 返回的用户名）
2. **前端**的 `request()` 能带上对应凭证（JWT `Authorization` 头 或 `X-User` 头）

当前项目所有权限判断都基于 `X-User` 中的用户名字符串，接入任何认证系统后，只需确保持久化到 `localStorage` 的用户名与后端识别的一致即可。
